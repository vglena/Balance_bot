# Registro de Decisiones

> Este archivo registra decisiones importantes sobre el sistema, la planificación o la vida cotidiana.
> Sirve para aprender de los patrones y no repetir errores.

---

## Formato de entrada

```
## [Fecha] — [Título breve de la decisión]
- **Situación:** Qué estaba pasando
- **Decisión:** Qué se decidió hacer
- **Razón:** Por qué se tomó esa decisión
- **Resultado:** Qué pasó (a rellenar después)
- **Aprendizaje:** Qué se haría diferente la próxima vez
```
