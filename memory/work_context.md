## Reglas de planificación del trabajo

- Proteger bloques de foco por la mañana cuando sea posible.
- Evitar trabajo profundo cerca de las recogidas de los niños.
- Reservar tareas ligeras para huecos cortos.
- No planificar entregables importantes después de las 20:30 salvo urgencia real.
- Separar trabajo urgente de trabajo importante.
- Si el día está saturado, elegir una única prioridad profesional.

## Tipos de bloque de trabajo

### Trabajo profundo
Para entregables, arquitectura, estrategia, escritura compleja o diseño de sistemas.

### Trabajo ligero
Para emails, revisión de notas, seguimiento, pequeñas correcciones o administración.

### Reuniones
Deben proteger margen antes y después si afectan a recogidas o logística familiar.

### Investigación
Debe tener objetivo claro para evitar dispersión.
