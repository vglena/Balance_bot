# Tareas domésticas

## Principio general

Las tareas domésticas deben planificarse en bloques pequeños y realistas.
El sistema debe evitar llenar el día con demasiadas tareas de casa si ya hay trabajo, recogidas o cansancio.

## Tipos de tareas habituales

- Comprar
- Limpiar
- Ordenar
- Poner lavadora
- Tender o recoger ropa
- Cocina
- Recoger cocina
- Basura
- Revisar nevera
- Preparar mochilas o cosas de los niños
- Pequeños recados

## Reglas de planificación doméstica

- Priorizar tareas imprescindibles antes que tareas deseables.
- Usar bloques de 10, 15, 20 o 30 minutos.
- No planificar limpieza grande en días saturados.
- Combinar recados con salidas si no añade estrés.
- Evitar tareas domésticas pesadas justo antes de las recogidas.
- Después de las 20:30 solo proponer tareas ligeras o cierre del día.
- Si hay poca energía, elegir una sola tarea doméstica mínima.

## Plan mínimo doméstico

En días complicados, el mínimo aceptable puede ser:

- Cocina razonablemente recogida
- Algo básico de ropa si es urgente
- Compra esencial si falta comida
- Preparar lo necesario para el día siguiente
