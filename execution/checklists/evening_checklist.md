# Checklist de Noche

Usar después de dormir a los niños (~20:30).

---

## Transición (20:30–21:00)
- [ ] Cenar (si no se ha cenado antes)
- [ ] 15-20 minutos de descompresión antes de ponerse a trabajar o planificar
- [ ] ¿Cuál es el nivel de energía ahora mismo?
  - [ ] Alta → se puede trabajar o planificar
  - [ ] Media → solo tareas ligeras
  - [ ] Baja → solo tareas de 5 min o directamente descanso

## Bloque nocturno (21:00–22:00, opcional)

Elegir UNA opción según energía y urgencias:

- [ ] Tarea urgente de trabajo (si hay deadline real)
- [ ] Preparar el día siguiente (rellenar `/context/current_day.md`)
- [ ] Tarea doméstica rápida (preparar ropa, recoger)
- [ ] Revisión semanal (si es viernes)
- [ ] Descanso consciente (lectura, serie, nada)

## Cierre del día
- [ ] Anotar en `/memory/session_notes.md` si hubo algo relevante hoy
- [ ] Preparar ropa para mañana (niños y adulto)
- [ ] Revisar si hay recados o cosas especiales mañana
- [ ] Apagar el ordenador antes de las 22:00

---

## Regla del bloque nocturno

Si la energía es baja, el descanso ES la decisión correcta.
No forzar trabajo cuando el cuerpo pide parar.
Mañana es otro día.
